# Adminer & Shell Links

Replace placeholders and open:

- Adminer (Postgres): `http://localhost:8080/?pgsql=postgres&username=postgres&db=testdb`
- Adminer (MySQL): `http://localhost:8080/?server=mysql&username=root&db=testdb`
- psql: `psql "host=127.0.0.1 port=5432 dbname=testdb user=postgres"`
- mysql: `mysql -h 127.0.0.1 -P 3306 -u root testdb`
